# T4.2 — Post-merge summary

This short note documents the observable, non-speculative changes merged as part of T4.2 and where to find them in the repository.

What was merged (observable in repository):

- HTTP schema layer using Zod under: apps/api/src/http/schemas/ (categories, products, orders, shared, index.ts)
- Schema registration plugin: apps/api/src/plugins/http-schemas.plugin.ts — converts Zod → JSON Schema via zod-to-json-schema and calls app.addSchema()
- Swagger plugin: apps/api/src/plugins/swagger.plugin.ts — registers @fastify/swagger and @fastify/swagger-ui and exposes UI in /docs
- Bootstrap wiring: apps/api/src/app.ts now calls registerSwagger(app) and registerHttpSchemas(app)
- Unit tests for schemas: apps/api/src/http/schemas/**tests**/schemas.spec.ts

Why this matters:

- Schemas are the single source of truth for HTTP validations (body/params/query/response) and are used to generate OpenAPI specs.
- The plugin registers schema components in Fastify so routes can reference them by $ref and Swagger UI shows consistent specs.
